# FocusFlow — Smart Productivity Dashboard

A premium, full-featured productivity web application built with vanilla HTML, CSS, and JavaScript.

## Features

- 📊 **Dashboard** — Overview of tasks, habits, focus sessions, and productivity score
- ✅ **Task Manager** — Create, edit, filter, search, and manage tasks with priorities and categories
- 🔄 **Habit Tracker** — Track daily habits with weekly views, streaks, and progress
- ⏱️ **Focus Timer** — Pomodoro timer with configurable focus/break durations and session history
- 📅 **Study Planner** — Weekly calendar, exams & goals, weekly targets, and revision reminders
- ⚙️ **Settings** — Customizable profile, goals, timer presets, accent themes, and data management

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Premium dark theme with glassmorphism, gradients, and animations
- **JavaScript** — Vanilla JS with localStorage persistence
- **Google Fonts** — Inter typeface

## Getting Started

Simply open `index.html` in your browser — no build tools or server required.
